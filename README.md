# RoWifi Alpha Modules

This repository contains sample modules for a Roblox verification bot built with **DSharpPlus**.

## Available Modules

- `Blacklists.cs` – Provides commands to manage user, group, and custom blacklists within a guild.

## Building

These files are provided as examples and are not part of a complete project. To compile them, integrate the module into your existing DSharpPlus bot project.
